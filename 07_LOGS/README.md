# 07_LOGS

`07_LOGS` is the evidence and operating-history layer.

Public Core includes index examples and log templates only. Real build logs, daily notes, agent activity, browser runs, workflow proofs, approvals, and runtime evidence are private deployment history.

## Starter Areas

- `Build-Logs/`
- `Daily/`
- `Agent-Activity/`
- `Workflow-Proofs/`
- `Operator-Briefs/`

## Private Deployment Rule

Logs can contain local paths, operational timing, project names, approval records, and runtime traces. Publish templates, not live logs.
