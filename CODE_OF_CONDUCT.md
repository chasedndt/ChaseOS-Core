# Code of Conduct — ChaseOS Core

We pledge a harassment-free experience for everyone. Be respectful and constructive;
accept feedback gracefully; focus on what is best for the project. Harassment,
personal attacks, publishing others' private information, and other inappropriate
conduct are not tolerated.

This applies in all project spaces and when representing the project publicly.
Report violations to conduct@chaseos.ai; maintainers may take any action they deem
appropriate. Adapted from the spirit of the Contributor Covenant.
