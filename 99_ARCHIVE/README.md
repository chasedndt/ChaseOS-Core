# 99_ARCHIVE

`99_ARCHIVE` stores historical records, documentation history, superseded material, handoffs, and prior-state evidence.

Public Core includes archive structure examples only. Real archive content from a live instance is private because it can contain personal history, old paths, project details, and operational decisions.

## Starter Areas

- `Documentation-History/`
- `Handovers/`
- `Snapshots/`

## Private Deployment Rule

Archive notes are not automatically safe just because they are old. Review before sharing.
