# Agent Control Plane

The Agent Control Plane defines how runtimes are registered, scoped, reviewed, and audited.

## Core Requirements

- Runtime identity is declared.
- Allowed task types are listed.
- Write authority is bounded.
- Approval requirements are explicit.
- Evidence is recorded for meaningful actions.

## Adapter Rule

Runtimes act through adapters. The control plane owns authority boundaries and review posture.
