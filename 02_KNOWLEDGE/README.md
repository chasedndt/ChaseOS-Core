# 02_KNOWLEDGE

`02_KNOWLEDGE` is the promoted knowledge layer.

Public Core includes synthetic examples that show the shape of source notes, synthesis notes, and knowledge indexes. Real research notes, personal learning records, client material, and sourced claims belong in the private instance.

## Starter Files

- `Knowledge-Index.example.md`
- `Source-Note.example.md`
- `Synthesis-Note.example.md`

## Private Deployment Rule

Keep raw inputs in `03_INPUTS` until reviewed. Promote only verified material into this layer.
