# 00_HOME

`00_HOME` is the operator home layer for a ChaseOS instance.

Use this folder for the current operating view, dashboard, principles, and fork-local orientation files. Public Core ships examples only. A private deployment should replace the examples with real operator truth.

## Public Core Contents

- `Operating-System.example.md`
- `Now.example.md`
- `Dashboard.example.md`
- `Principles.example.md`

## Private Deployment Rule

Do not publish populated operator state, live sprint truth, personal doctrine, account details, or current project facts from this folder.
