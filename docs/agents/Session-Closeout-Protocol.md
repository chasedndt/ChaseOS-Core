# Session Closeout Protocol

Closeout records make long-running work recoverable.

## Closeout fields

- work completed;
- artifacts created;
- tests/verifications run;
- decisions made;
- unresolved blockers;
- next safe action;
- authority/gate status.

## Do not include

- secrets;
- raw private data;
- unverified success claims;
- instructions that bypass approval gates.
